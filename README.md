# 🏠 House Price Prediction

A Machine Learning web application that predicts house prices based on user inputs such as city, property type, BHK, bathroom count, parking availability, and total area. The project uses a trained regression model and a Flask web application to provide real-time price predictions.

---

## 📌 Project Overview

This project demonstrates an end-to-end Machine Learning workflow including:

- Data Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation
- Model Serialization
- Flask Web Application
- Real-time Predictions

The objective is to estimate house prices accurately based on important property features.

---

## 🚀 Features

- Predict house prices instantly
- User-friendly Flask web interface
- Data preprocessing using Scikit-learn
- Machine Learning Regression model
- Clean and responsive UI
- Real-time prediction results
